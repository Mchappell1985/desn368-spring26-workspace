# Mac Chappell Portfolio

This repository contains a web design portfolio home page, a course learning log, and archived coursework from the web design class.

## What is included

- `index.html` — Portfolio homepage
- `css/style.css` — Portfolio styling
- `learning-log.html` — Learning log page with weekly progress and project links
- `archive/course-log.md` — Original course README content moved into archive
- `Week-1/` through `Week-9/` — Weekly assignments and project files
- `images/` — Shared images used by the portfolio and coursework
- `styles/` — Existing project styles from course work

## How to view

1. Open `index.html` in a browser.
2. Use the navigation links to browse the learning log and featured projects.

## Notes

- The original course README has been preserved at `archive/course-log.md`.
- All internal links in `learning-log.html` remain intact.
